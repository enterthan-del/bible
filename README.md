# 성경 암송 자료 모바일 앱

44개 핵심 성경 구절을 쉽게 암송할 수 있는 모바일 앱입니다.

## 📦 포함된 파일

- **index.html** - 메인 앱 파일 (반드시 필요)
- **manifest.json** - PWA 설정 파일 (반드시 필요)
- **service-worker.js** - 오프라인 지원 (반드시 필요)
- **icon-generator.html** - 앱 아이콘 생성 도구
- **앱_설치_가이드.md** - 상세 설치 및 사용 가이드

## 🚀 빠른 시작

### 1단계: 아이콘 생성 (선택사항)
1. `icon-generator.html` 파일을 브라우저에서 열기
2. 두 개의 아이콘 이미지 다운로드
3. 다운로드한 파일을 같은 폴더에 저장

### 2단계: 웹 호스팅
파일들을 웹 서버에 업로드하세요:
- **추천**: GitHub Pages, Netlify, Vercel (무료)
- **요구사항**: HTTPS 연결 필수

### 3단계: 모바일에 설치
1. 업로드한 URL을 모바일 브라우저에서 접속
2. **iPhone**: Safari → 공유 → 홈 화면에 추가
3. **Android**: Chrome → 메뉴 → 앱 설치

## 📱 기능

✅ 44개 핵심 성경 구절  
✅ 심플한 삽화와 시각적 학습  
✅ 첫 글자 힌트 + 암기 문장  
✅ 터치 스와이프 제스처  
✅ 오프라인 사용 가능  
✅ iOS & Android 지원
