# 성경 암송 자료 모바일 앱 설치 가이드

## 📱 앱 개요
44개 핵심 성경 구절을 쉽게 암송할 수 있도록 도와주는 모바일 앱입니다.
- 심플한 삽화와 함께 성경 구절 학습
- 첫 글자 힌트와 암기 문장 제공
- 오프라인에서도 사용 가능
- iOS와 Android 모두 지원

---

## 📲 설치 방법

### 방법 1: PWA (Progressive Web App) 설치 - **가장 추천!**

#### iPhone/iPad (iOS/iPadOS)
1. Safari 브라우저에서 웹사이트 접속
2. 화면 하단의 공유 버튼 탭 (□↑ 아이콘)
3. "홈 화면에 추가" 선택
4. "추가" 탭하여 설치 완료
5. 홈 화면에 생성된 아이콘으로 앱 실행

#### Android
1. Chrome 브라우저에서 웹사이트 접속
2. 우측 상단 메뉴(⋮) 탭
3. "홈 화면에 추가" 또는 "앱 설치" 선택
4. 확인하여 설치 완료
5. 홈 화면이나 앱 서랍에서 실행

---

### 방법 2: 웹 호스팅 후 접속

#### 필요한 파일들:
```
📁 앱 파일
├── index.html              (메인 앱 파일)
├── manifest.json          (PWA 설정)
├── service-worker.js      (오프라인 지원)
└── (선택) icon-192.png, icon-512.png (앱 아이콘)
```

#### 호스팅 방법:

**A. GitHub Pages (무료, 추천)**
1. GitHub 계정 생성 (github.com)
2. 새 저장소(repository) 생성
3. 파일들을 저장소에 업로드
4. Settings → Pages에서 배포 활성화
5. 제공된 URL로 접속

**B. Netlify/Vercel (무료, 쉬움)**
1. Netlify.com 또는 Vercel.com 접속
2. 파일들을 드래그 앤 드롭으로 업로드
3. 자동 배포 완료
4. 제공된 URL로 접속

**C. 개인 서버/호스팅**
- 파일들을 웹 서버에 업로드
- HTTPS 연결 필수 (PWA 요구사항)

---

### 방법 3: React Native로 네이티브 앱 개발 (고급)

본격적인 앱 개발을 원하시면:
1. React Native 프로젝트 생성
2. 현재 HTML/JS 로직을 React Native 컴포넌트로 변환
3. Expo 또는 React Native CLI로 빌드
4. App Store/Google Play Store 배포

**참고**: 이 방법은 개발 경험이 필요합니다.

## 🎯 사용 방법

### 기본 조작
- **탭/클릭**: 다음 줄 표시
- **스와이프 왼쪽**: 다음 슬라이드
- **스와이프 오른쪽**: 이전 슬라이드
- **◀ 이전/다음 ▶ 버튼**: 슬라이드 이동

### 슬라이드 구성
- **힌트 슬라이드**: 첫 글자만 표시 + 암기 문장
- **전체 슬라이드**: 성경 구절 전체 본문

### 학습 팁
1. 힌트 슬라이드를 먼저 보고 기억해보기
2. 암기 문장(💡)을 활용하여 순서 기억
3. 전체 슬라이드로 확인
4. 반복 학습으로 암송 완성

## 🔧 문제 해결

### PWA 설치가 안 될 때
- HTTPS 연결 확인 (HTTP는 불가)
- 브라우저가 PWA를 지원하는지 확인
- 시크릿/프라이빗 모드가 아닌지 확인

### 오프라인에서 작동하지 않을 때
- 첫 방문 시 온라인 상태에서 한 번 로딩 필요
- 브라우저 캐시 삭제 후 다시 접속

### 아이콘이 표시되지 않을 때
- icon-192.png, icon-512.png 파일 생성 필요
- 또는 SVG 파일을 PNG로 변환하여 사용

---

## 📝 추가 개선 사항 (선택)

1. **앱 아이콘 이미지 생성**
   - icon-192.svg를 PNG로 변환
   - 온라인 도구: favicon.io, cloudconvert.com

2. **커스터마이징**
   - manifest.json에서 앱 이름, 색상 변경
   - HTML에서 구절 추가/수정 가능

3. **고급 기능 추가**
   - 학습 진도 저장 (localStorage)
   - 즐겨찾기 기능
   - 검색 기능
